## Color Psychology

### Browns (Primary)
- **Feeling**: Professional, warm, reliable, grounded
- **Industry fit**: Content creation, marketing, consulting
- **Avoids**: Tech clichÃ©s (blue), overly playful (bright colors)

### Grays (Neutral)
- **Feeling**: Clean, modern, sophisticated
- **Industry fit**: Universal, professional
- **Avoids**: Sterile (pure grays use warm undertones)

### Semantic Colors
- **Green**: Growth, success, completion
- **Amber**: Attention, caution, in-progress
- **Red**: Error, urgency, critical
- **Blue**: Information, trust, neutrality

## Print & Export

When exporting or printing:
- Use light mode colors
- Ensure sufficient contrast (black text on white)
- Avoid colored backgrounds (save ink)
- Use semantic colors sparingly

---

This color system ensures consistency, accessibility, and professional aesthetics across both light and dark themes.
